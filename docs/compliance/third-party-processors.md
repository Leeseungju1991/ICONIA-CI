## 4. 위탁/제3자 변경 시

- 추가 시: 사용자에게 사전 고지 + (필수 동의 변경 시) 재동의
- 제거 시: 사용자에게 공지
- 위탁 업무 변경 시: 사용자에게 공지

## 5. 책임

회사는 위탁받은 사업자에 대해 다음을 이행한다.

- 위탁 계약서 체결 (개인정보 보호법 제26조 제2항)
- 위탁 사업자의 안전성 확보 조치 점검
- 위탁 사업자의 개인정보 보호 정책 검토
- 정기적 위탁 사업자 감사
