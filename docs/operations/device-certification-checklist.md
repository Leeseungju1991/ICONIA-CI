# Device Certification Checklist (HW)

> ICONIA 인형 (ESP32-S3 기반) 의 양산·판매 전 인증 점검표.
> **출시 전 외부 인증 전문가 검토 필수** (HD-03).

## 1. 적용 인증 (잠정)

| 인증 | 적용 여부 | 사유 | 담당 |
|---|---|---|---|
| KC 적합성평가 (방송통신기자재) | **적용** (BLE/Wi-Fi 송수신) | 전파법 + 방송통신기자재 등의 적합성평가에 관한 고시 | 외부 인증 기관 (전파연구원) |
| KC 안전인증 (전기용품) | (조건부) 어댑터/충전기 동봉 시 | 전기용품 및 생활용품 안전관리법 | 외부 인증 기관 |
| KC 안전·확인 (어린이 제품) | (해당 없음) 성인 대상 제품 | 어린이제품 안전 특별법 | — |
| RoHS (전자제품 유해물질) | 권장 | EU/일반 수출 시 필수 | 외부 시험기관 |
| REACH | (해당 시) 화학물질 — 인형 재질 | EU 수출 시 | 외부 시험기관 |

| CE (유럽) | (해외 출시 시) | EU 시장 진입 | NB |

## 2. KC 적합성평가 (방송통신기자재) 진행 절차

### 2.1 사전 점검 (양산 직전)
- [ ] BLE/Wi-Fi 모듈 (ESP32-S3) 자체가 KC 인증된 모듈인지 확인 — **YES** (Espressif 모듈은 한국 KC 인증 보유)
- [ ] 모듈 차용 시 추가 적합성평가 필요 여부 확인 — 외관/케이스 변경 시 재평가
- [ ] EMC (전자파 적합성) 시험
- [ ] EMI (전자파 방사) 시험
- [ ] 안전 시험 (배터리 사용 시)

### 2.2 시험 항목
- [ ] 무선 출력 파워 (BLE)
- [ ] 무선 출력 파워 (Wi-Fi 2.4GHz)
- [ ] 점유 주파수 대역
- [ ] 스퓨리어스 방사
- [ ] 안테나 성능
- [ ] 일반 EMC 시험 (전자파 양립성)

### 2.3 서류 준비
- [ ] 제품 사양서
- [ ] 회로도 (BLE/Wi-Fi 안테나 부 포함)
- [ ] 외관 사진 (전·후·측면)
- [ ] 라벨 도안 (KC 마크 + 인증번호 배치)
- [ ] 제조사 정보 ((주)숨코리아)
- [ ] 수입자 정보 (해당 시)

### 2.4 인증 결과 반영
- 인증번호 → 앱 [기기 인증 정보] 화면 표시
- 인증번호 → 제품 라벨/매뉴얼 표시
- 인증번호 → 본 checklist 의 "결과" 섹션 갱신

## 3. 안전 인증 (전기용품)

### 3.1 적용 여부 결정
- 어댑터 동봉 → 별도 KC 안전인증 필요 (어댑터 자체가 인증된 시판품 사용 시 회사가 별도 인증 불필요)
- USB-C 충전 케이블 동봉 → 일반 시판품 (인증 불필요)
- 배터리 (Li-ion/Li-Po) 내장 → 배터리 자체 KC + 제품 종합 안전 시험

### 3.2 시험 항목 (배터리)
- [ ] 과충전/과방전 보호
- [ ] 과전류 보호
- [ ] 단락 보호
- [ ] 온도 보호
- [ ] 진동/충격 안전성
- [ ] 압력 안전성
- [ ] 외관 안전성 (날카로운 모서리 등)

## 4. RoHS / REACH (선택, 수출 시 권장)

- [ ] 납 (Pb) — 0.1% 이하
- [ ] 카드뮴 (Cd) — 0.01% 이하
- [ ] 수은 (Hg) — 0.1% 이하
- [ ] 6가 크로뮴 (Cr⁶⁺) — 0.1% 이하
- [ ] PBB, PBDE — 각 0.1% 이하

## 5. 양산 라인 검증 (production_test)

펌웨어의 `production_test` 모드를 활용한 양산 라인 LOT 별 yield 집계:
- BLE 송수신 자가진단
- Wi-Fi 송수신 자가진단
- 배터리 충전/방전 통과 여부
- 센서 calibration

`6. CI/docs/operations/`에서 정의된 양산 procedure 와 연결.
