(주)숨코리아 / Soom Korea Inc. (이하 "회사")는 ICONIA 서비스(이하 "서비스") 이용자의 개인정보를 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 등 관련 법령에 따라 보호합니다.

## 1. 수집하는 개인정보 항목 및 수집 방법

### 1.1 회원가입 시 (필수)

| 항목 | 수집 방법 | 비고 |
|---|---|---|
| 이메일 주소 | 사용자 직접 입력 | 로그인 ID + 통신 수단 |
| 비밀번호 | 사용자 직접 입력 | bcrypt 단방향 해시로만 저장 (평문 비저장) |
| 이용약관 동의 이력 | 동의 화면 | 동의 시각·버전·IP·UA 기록 |
| 개인정보 수집·이용 동의 이력 | 동의 화면 | 동일 |
| 만 19세 이상 본인 확인 | 자가 확인 체크 | 본 서비스는 성인 대상 제품 |

### 1.2 회원가입 시 (선택)

| 항목 | 수집 시점 | 거부 시 영향 |
|---|---|---|
| 표시 이름 (display name) | 가입/설정 | 익명 표시 |
| 마케팅 정보 수신 동의 | 가입/설정 | 마케팅 알림 미수신 |
| 제3자 제공 동의 (Google Gemini) | 가입 | AI 페르소나 응답 사용 불가 |
| 국외이전 동의 (Gemini US region) | 가입 | AI 페르소나 응답 사용 불가 |
| 분석·서비스 개선 동의 | 가입/설정 | 익명 분석 미활용 |

### 1.3 서비스 이용 중 자동 수집

| 항목 | 목적 | 익명화 |
|---|---|---|
| 디바이스 식별자 (UUID) | 페어링·재연결 | 페어링 해제 시 익명화 |
| 펌웨어 버전, 마지막 접속 시각, 배터리 잔량 | 운영 모니터링 | — |
| AI 대화 내용 | 페르소나 응답 + 컨텍스트 | PII 마스킹 후 저장 |
| 피드 게시물·댓글 | 사용자 발화 — 사용자 동의 하 게시 | 사용자 삭제 즉시 익명화 |
| 접속 IP, 기기 정보, 요청 ID | 장애 대응·보안 | 30일 후 익명화 |

### 1.4 디바이스(인형) 측 수집·송신 데이터 (정본: `1. HW/docs/legal/data-collection.md`)

| 항목 | 디바이스 측 보관 | 회사 서버로의 송신 |
|---|---|---|
| 카메라 캡처 JPEG (사용자 손길 터치 시 1장) | RAM only — 업로드 직후 즉시 폐기 | TLS 1.2 + ACM 인증서 송신 → AWS S3 (서울) |
| 터치 이벤트 (좌/우/타이밍) | RTC slow-memory (중복 방지 키) | 이벤트 메타와 함께 송신 |
| 배터리 잔량·상태 (6단계) | NVS 마지막 값 | 이벤트 메타와 함께 송신 |
| Wi-Fi SSID / 비밀번호 | NVS (AES-256-GCM 암호화) | **회사 서버에 송신 안 함** — 디바이스 로컬 only |
| 디바이스 식별자 (device_id_hash) | NVS / RTC | 송신 (페어링 + 이벤트) |
| 부팅·OTA·에러 telemetry | NVS / RTC 큐 | 송신 (운영 모니터링) |

### 1.5 디바이스에서 **수집하지 않는** 데이터

- 위치 정보 (GPS / Wi-Fi 측위 — 펌웨어에 측위 로직 없음)
- 마이크/음성 (마이크 미탑재)
- 연속 영상/항시 녹화 (Deep Sleep 정책 — 손길 터치 시 1장만)
- 결제·금융 정보 (앱 내 결제 미제공)
- 생체 정보 (얼굴 인식·지문 인식 — 펌웨어 측 미수행)
- 위치 기반 광고 식별자 (IDFA/AAID)

### 1.6 운영자 (Admin Console)

운영자 계정 (회사 직원) 의 이메일·비밀번호 해시·MFA(TOTP) 시크릿·세션·활동 감사 로그를 수집합니다. 운영자 권한 정책은 `docs/operations/admin-operation-policy.md` 정본.

## 2. 개인정보의 이용 목적

- 회원 가입·인증·관리
- AI 인형 페어링·페르소나 응답 제공
- 서비스 운영 (장애 대응·보안 모니터링·부정 이용 차단)
- 법령상 의무 이행 (동의 이력 5년 보관, 개인정보 접근 통제 5년 보관)
- (동의 시) 마케팅·이벤트 정보 제공

## 3. 개인정보의 보관 기간

| 데이터 | 보관 기간 | 보관 후 처리 |
|---|---|---|
| 회원 가입 정보 (이메일·display_name) | 회원 탈퇴 시 즉시 익명화 + 24개월 후 완전 폐기 | DELETE row |

| 접속 로그 (개인정보 접근) | 5년 (개인정보 보호법 안전성 확보 조치) | DELETE row |

| 디바이스 데이터 (식별자·진단) | 페어링 해제 시 익명화 + 24개월 후 완전 폐기 | — |
| 카메라 JPEG | AWS S3 보관 정책에 따름 — `docs/compliance/data-retention-policy.md` 정본 | — |

회사는 이용자의 사전 동의 없이 개인정보를 제3자에게 제공하지 않습니다.

| 제공 대상 | 제공 데이터 | 제공 목적 | 동의 항목 |

거부 시 AI 페르소나 응답 기능 이용이 제한됩니다.

권리 행사 결과는 「개인정보 보호법」 제35조~제37조에 따라 처리됩니다.

## 8. 개인정보 안전성 확보 조치

- 비밀번호 단방향 암호화 (bcrypt)
- 모든 통신 TLS 1.2 이상 (AWS ACM)
- 디바이스 ↔ 서버 통신 SPKI 핀 적용 (선택)
- 운영자 콘솔 2단계 인증 (TOTP) 필수
- 운영자 개인정보 접근 감사 로그 (해시체인 무결성 + 5년 보관)
- AWS Secrets Manager / IAM 최소 권한 원칙
- 정기 backup·복구 절차 — `docs/operations/backup-restore-runbook.md`
- 침해사고 대응 절차 — `docs/operations/personal-data-breach-runbook.md`

## 9. AI 응답 관련 안내
