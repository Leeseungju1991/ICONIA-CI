# 디바이스 연결·데이터 안내

## 1. ICONIA 인형 안내

ICONIA 인형은 블루투스 5.0 + Wi-Fi 2.4GHz 무선 통신 기능을 탑재한 IoT 디바이스입니다.

| 항목 | 값 |
|---|---|
| 무선 통신 | 블루투스 5.0 + Wi-Fi 2.4GHz |
| 카메라 | 사용자 손길 터치 시 1장 캡처 (임시 메모리에서만 처리, 업로드 후 즉시 폐기) |
| 마이크 | 미탑재 |
| 위치 기능 | 미탑재 |
| 인증 | KC 적합성평가 (양산 출하 전 진행) |

## 2. 페어링 및 프로비저닝

ICONIA 앱과 인형을 연결하기 위해 다음 절차를 거칩니다.

1. **블루투스 권한 동의** — 앱이 인형을 블루투스로 검색·연결하기 위해 필요합니다.
2. **위치 권한 (Android 만)** — Android OS 의 블루투스 스캔은 운영체제 정책상 위치 권한을 함께 요구합니다. **인형의 위치나 사용자의 위치를 수집하지 않습니다.** 위치 권한은 운영체제 정책상의 형식적 요건입니다.
3. **Wi-Fi 비밀번호 입력** — 인형이 가정 Wi-Fi 에 연결하기 위해 사용자의 Wi-Fi 이름과 비밀번호를 블루투스 연결을 통해 인형에 직접 전송합니다.

### Wi-Fi 비밀번호 저장 정책

- **앱**: 사용자가 입력한 Wi-Fi 비밀번호는 사용자 휴대전화의 보안 저장소에 암호화하여 저장합니다 (사용자 디바이스 한정).
- **인형**: 블루투스 연결을 통해 수신한 Wi-Fi 비밀번호는 인형 내부 저장소에 암호화하여 저장합니다.
- **회사 서버**: Wi-Fi 비밀번호는 **회사 서버에 전송되거나 저장되지 않습니다.**

## 5. 디바이스 안전 사용 안내

- 디바이스를 **분해·개조**하지 마세요. 보증 및 안전 인증이 무효화됩니다.
- 디바이스가 **과열·연기·이상 소음**을 보일 경우 즉시 전원을 차단하고 회사 고객지원(web@soomkorea.com)에 문의하세요.
- 어린이의 손에 닿지 않는 곳에 보관하세요 (본 제품은 성인 대상 제품입니다).
- 디바이스는 가정 내 사용을 전제로 설계되었습니다. 야외·자동차 내부·고온 환경(60℃ 이상) 등에서 사용하지 마세요.
- 배터리 충전 시 정품 충전기·케이블만 사용하세요.

## 6. 기기 소프트웨어 자동 업데이트 안내

- 회사는 보안·기능 개선을 위해 정기적으로 디바이스 소프트웨어를 자동으로 업데이트할 수 있습니다.
- 업데이트는 자동으로 진행되며, 사용자가 거부할 경우 일부 기능 사용이 제한될 수 있습니다.
- 업데이트는 보안을 위해 이전 버전으로 임의 복구하지 않는 정책을 따릅니다.
- 업데이트 실패 시 자동으로 이전 안정 버전으로 복귀하며, 사용자 데이터는 영향받지 않습니다.

4. 진단 정보 삭제
5. 세션 정보 초기화
6. 회사 고객지원 채널에 데이터 삭제 완료 보고
