# AI 이용 안내 / 면책 정책

## 1. AI 페르소나 사용 안내

ICONIA 서비스의 AI 페르소나는 다음 기술을 사용합니다.

- **Google Gemini API** — 응답 생성을 위한 대규모 언어 모델
- **자체 페르소나 엔진** — 페르소나 일관성, 욕구·신호 매핑, 기억 관리, 감정 표현 등 회사 자체 기술
- **검색 기반 응답** — 페르소나 별 기억·문서를 검색해 응답에 활용

## 2. 사용자에게 표시되는 안내 문구 (앱 내 [AI 이용 안내] 화면)

> ICONIA 의 AI 페르소나 응답은 인공지능이 생성한 것입니다.
>
> - 응답은 사실과 다르거나 부정확할 수 있으며, 의료·법률·금융 등 전문가 상담이 필요한 분야의 결정 근거로 사용하지 마세요.
> - 비밀번호·카드번호·주민등록번호 등 민감한 개인정보를 AI 와의 대화에 입력하지 마세요.
> - AI 페르소나 응답은 일관성을 위해 캐릭터화되어 있으며, 실제 인물이나 의료적·법률적 진단을 대체하지 않습니다.
> - AI 응답이 부적절하다고 판단되시면 [신고하기] 를 눌러 회사에 알려주세요.

## 3. 데이터 처리

| 데이터 | 처리 위치 | 보관 기간 | 마스킹 |
|---|---|---|---|
| 사용자 입력 메시지 | Google Gemini (미국) + 회사 클라우드 데이터베이스 (대한민국) | 회사 보관 90일 / Google 은 Google 정책에 따름 | 개인정보 마스킹 후 저장·송신 |
| AI 응답 | 회사 클라우드 데이터베이스 (대한민국) | 90일 | 그대로 보관 |
| 페르소나 컨텍스트 (기억) | 회사 클라우드 저장소 (대한민국) | 사용자 탈퇴 시 폐기 | 사용자별 격리 |
| 운영 로그 | 보안이 적용된 클라우드 로그 (대한민국) | 30일 | 자동 개인정보 마스킹 |

## 4. 사용자 동의 항목

회원 가입 시 다음 동의를 받습니다.

- **(필수) 제3자 제공 동의 (Google Gemini)** — AI 응답을 위해 사용자 메시지를 Google 에 전송
- **(필수) 국외이전 동의** — 미국 지역으로 데이터 전송 (Gemini API)
- **(선택) 분석·서비스 개선 동의** — 익명 통계로 활용

## 5. 학습 데이터 사용

**회사 정책**:
- 회사는 사용자 대화 내용을 자체 AI 학습에 사용하지 **않습니다**.
- 사용자 대화는 Gemini API 호출 시 Google 의 정책에 따라 처리됩니다.
- 회사는 개인정보 마스킹을 적용한 후에만 데이터를 저장·송신합니다.

**자체 학습 데이터 활용은 현재 시행하지 않습니다.** 자체 학습을 시작하게 될 경우 다음 절차를 거칩니다.
- 별도 동의 항목 추가 (필수 또는 선택) — 별도로 동의를 받습니다.
- 사용자가 동의 거부 시에도 기본 AI 기능은 정상 이용 가능
- 별도 [개인정보 처리방침] 및 [AI 이용 안내] 갱신 → 재동의

## 6. AI 안전성 조치

| 항목 | 조치 |
|---|---|
| 악의적 입력 방어 | 입력값 정제 + 모니터링 신호 적용 |
| 시스템 지시문 보호 | 응답 검증 + 누출 감지 시 자동 회수 |
| 부적절한 응답 차단 | Google 안전 필터 + 회사 자체 검증 |
| 개인정보 마스킹 | 외부 모델 호출 전 마스킹 처리 |
| 사실성 보강 | 검색 기반 사실 응답 우선, 모를 때 "확실하지 않다" 응답 |
| 페르소나 일관성 | 자체 페르소나 엔진 + 응답 후 일관성 검증 |
| 부정 사용 차단 | 요청 한도 제한 + 사용자 단위 이상 행동 감지 |

## 7. AI 장애 시 대응

| 시나리오 | 사용자 측 동작 | 운영팀 동작 |
|---|---|---|
| Gemini API 응답 지연 | 페르소나 일관성을 유지한 대체 응답 표시 | 외부 API 키 상태 점검 |
| Gemini API 요청 한도 초과 | 여러 API 키 순환 사용 + 쿨다운 | 신규 키 발급 또는 사용 등급 승급 |
| Gemini API 장애 | 페르소나 일관성 유지 + "응답이 잠시 지연됩니다" 안내 | 회사 내부 장애 대응 절차 진행 |
| 부적절한 응답 감지 | 응답 차단 + 페르소나에 어울리는 사과 메시지 | 응답 검토 + 페르소나 조정 |
| 모든 외부 API 키 쿨다운 | 대체 응답 + "AI 페르소나가 잠시 휴식 중입니다" 안내 | 즉시 신규 키 추가 + 사용 등급 승급 검토 |

## 8. 책임 제한

- AI 응답은 사용자의 의사결정을 대체하지 않습니다.
- 회사는 AI 응답으로 인한 직접/간접 손해에 대해 고의 또는 중과실이 없는 한 책임을 지지 않습니다.
- 사용자는 AI 응답을 비판적으로 검토하시고, 중요한 결정은 전문가 상담을 받으시기 바랍니다.

## 9. 신고 및 피드백

AI 응답이 부적절하거나 정책에 어긋난다고 판단되시면 [신고하기] 버튼으로 회사에 알려주세요. 회사는 신고 접수 후 영업일 2일 이내 검토하고 결과를 알려드립니다.

## 10. 사용자 권리

- AI 학습 데이터 사용 동의 철회 (현재 자체 학습 미사용)
- 제3자 제공 동의 철회 → AI 페르소나 응답 기능 비활성
- AI 대화 내역 열람 → 앱 설정 > AI 기록
- AI 대화 내역 삭제 → 앱 설정 > AI 기록 관리 > 전체 삭제 (회사 보관 기간 90일보다 빠르게 삭제)
