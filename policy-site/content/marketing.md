# 마케팅 정보 수신 동의

> 💡 **핵심 요약** — 마케팅 수신은 선택 항목입니다. 동의하지 않아도 기본 서비스는 정상 이용 가능합니다. 동의 후에도 언제든지 설정에서 철회할 수 있습니다.

## 1번 항목 — 동의 항목 (선택)

| 항목 | 내용 | 비고 |
|---|---|---|
| 이메일 마케팅 수신 | 서비스 안내·이벤트·할인 정보 | 선택 |
| 푸시 알림 마케팅 수신 | 동일 | OS 푸시 권한 + 본 동의 모두 필요 |
| 인앱 메시지 마케팅 | 동일 | 본 동의 시에만 표시 |
| SMS 마케팅 | 동일 | 현재 제공하지 않습니다. 시작 시 별도 동의를 받습니다 |

## 2번 항목 — 동의 문구

> **(선택) 마케팅 정보 수신 동의**
>
> 회사가 운영하는 ICONIA 서비스의 새 기능·이벤트·할인·뉴스 등을 이메일·푸시 알림·인앱 메시지로 받기 동의합니다.
>
> - 동의하지 않으셔도 기본 서비스는 정상 이용 가능합니다.
> - 동의 후 언제든지 [설정 > 마케팅 수신] 또는 수신한 메시지의 [수신거부] 링크로 동의를 철회할 수 있습니다.

## 3번 항목 — 동의 시점

- **회원 가입 시**: 선택 동의 화면에서
- **설정 화면에서**: 언제든지 동의·철회

## 4번 항목 — 동의 이력 보관

- 동의/철회 시각, 동의 항목, 접속 정보를 5년간 기록 보관
- 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 제50조(영리 목적의 광고성 정보 전송 제한)의 동의 입증 자료

## 5번 항목 — 광고성 정보 전송 규정 (정보통신망법 제50조)

| 항목 | 회사 조치 |
|---|---|
| 사전 동의 | 1번 항목 동의 후에만 전송 |
| 광고 명시 | 메시지 제목/본문에 "(광고)" 명시 |
| 무료 수신거부 수단 | 모든 메시지에 [수신거부] 링크 제공 |
| 야간 시간 제한 | 21:00~08:00 마케팅 푸시 전송 안 함 (긴급 공지 제외) |
| 철회 시 즉시 중단 | 철회 즉시 발송 큐에서 제외 |

## 6번 항목 — 마케팅 푸시 발송 운영

- 운영자가 관리자 페이지에서 푸시 캠페인 작성
- 발송 대상: 마케팅 수신 동의자만 자동 필터 (미동의자 자동 제외)
- 발송 전 운영자 확인 절차 필수
- 발송 결과 5년간 기록 보관

## 7번 항목 — 동의하지 않은 사용자 처리

- 마케팅 수신에 동의하지 않은 사용자는 발송 대상에서 자동 제외됩니다.
- 발송 시점의 동의 여부를 확인하여 처리합니다.

## 8번 항목 — 마케팅 항목 추가 시 절차

새로운 마케팅 항목(예: SMS, 외부 채널 광고 등)을 도입할 때에는 다음 절차를 따릅니다.

1. 사용자에게 재동의 요청
2. 동의 이력 신규 추가
3. 거부 사용자는 신규 항목 비활성 유지
