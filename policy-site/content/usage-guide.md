# 앱 사용 가이드

> 💡 **핵심 요약** — 인형 개봉 → 앱 설치 → 블루투스 페어링 → Wi-Fi 연결 → 일상 사용. 5단계 순서대로 따라하면 누구나 시작할 수 있어요. 평균 소요 시간 약 7분.

## 1번 항목 — 준비물 체크

- ICONIA 인형 (택배 박스 안)
- Android 스마트폰 (iOS는 출시 준비 중)
- 사용할 공간의 Wi-Fi 비밀번호
- 충전된 인형 또는 USB-C 충전 케이블

> ℹ️ 인형이 완전히 방전된 상태라면 30분 정도 먼저 충전한 뒤 시작하세요. LED 가 천천히 호흡하듯 깜빡이면 충전 중 상태입니다.

## 2번 항목 — 인형 개봉 (1분)

박스를 열고 인형을 조심스럽게 꺼냅니다. 인형 뒤쪽 또는 발 아래에 위치한 **전원 버튼을 3초간 길게** 누르면 LED 가 청색으로 깜빡이며 "페어링 대기" 상태로 들어갑니다.

- LED 청색 깜빡임 → 페어링 대기 (정상)
- LED 빨강 점멸 → 배터리 부족, 충전 필요
- LED 꺼짐 → 전원이 켜지지 않음, 충전 후 재시도

## 3번 항목 — 앱 설치 (2분)

홈 화면의 **[앱 다운로드]** 버튼을 누르면 `iconia-latest.apk` 파일이 다운로드 폴더에 저장됩니다 (약 40 MB).

| 단계 | 내용 |
|---|---|
| 1 | 다운로드 알림에서 "열기" 또는 다운로드 폴더에서 APK 탭 |
| 2 | "출처를 알 수 없는 앱 설치" 한 번만 허용 |
| 3 | "설치" 버튼 → 자동 진행 (10초) |
| 4 | "열기" 또는 홈 화면의 ICONIA 아이콘으로 앱 실행 |

> ⚠️ 처음 설치 시 한 번만 권한 허용이 필요합니다. 보안 상 다른 앱 설치는 다시 차단되니 안심하세요.

## 4번 항목 — 블루투스 페어링 (1분)

앱을 열고 회원가입(이메일 1분 소요)을 완료한 뒤 **[인형 연결]** 버튼을 누릅니다. 주변에 켜진 인형이 자동으로 검색돼 목록에 표시됩니다.

- 인형 이름을 탭하면 페어링이 시작 (3-5초)
- 페어링이 끝나면 인형 LED 가 보라색으로 한 번 깜빡임
- 앱에서 페르소나 이름을 정해 줄 수 있어요

## 5번 항목 — Wi-Fi 연결 (1분)

페어링 직후 앱이 **Wi-Fi 정보 입력 화면**으로 이동합니다.

- 집 또는 공간의 Wi-Fi 이름(SSID)을 선택
- 비밀번호 입력 → "연결" 탭
- 인형 LED 가 초록색으로 바뀌면 연결 완료

> ℹ️ Wi-Fi 비밀번호는 **인형 내부에만 암호화 저장**되며 회사 서버로는 전송되지 않습니다. 자세한 내용은 [기기 연결 안내](#/device) 참조.

## 6번 항목 — 일상 사용

연결이 끝나면 인형이 평소 상태(LED 은은하게 점등) 로 들어갑니다. 인형의 **머리·이마·얼굴**을 살짝 만지면 카메라가 사진 1장을 찍어 앱으로 전달하고, AI 페르소나가 짧은 인사를 건넵니다.

- 손길 1회당 사진 1장 (반복 터치 시 마지막 1장만 전달)
- 앱에서 페르소나 톤·관계 설정 가능
- 하루 사용량은 앱 [내 정보] 에서 확인

> ✅ 인형은 상시 촬영하지 않습니다. **사용자가 만질 때만** 셔터가 동작해요.

## 7번 항목 — 자주 묻는 질문 (FAQ)

| 질문 | 해결 방법 |
|---|---|
| 인형 LED 가 빨갛게 깜빡여요 | 배터리 부족 → USB-C 충전 케이블 연결 |
| Wi-Fi 가 안 잡혀요 | 앱 [기기 설정] → [Wi-Fi 다시 연결] |
| 앱이 인형을 못 찾아요 | 인형 전원 OFF→ON, 앱 블루투스 권한 확인 |
| 사진은 어디 저장되나요? | 인형 안에는 저장 안 됨. 회사 서버(한국 리전, 암호화)에 보관 후 90일 뒤 익명화 |
| AI 가 이상한 말을 해요 | 앱 [신고] 또는 web@soomkorea.com 으로 문의 |
| 인형을 다른 사람에게 넘기려고 해요 | 앱 [기기 설정] → [공장 초기화]. 자세한 절차는 [기기 연결 안내](#/device) 7번 항목 참조 |
