# 위치정보 이용 정책

> **현재 본 서비스는 위치정보를 수집·이용하지 않습니다.** 향후 위치 기반 기능(예: 매장 안내, 디바이스 분실 추적) 도입 시 본 정책이 활성화됩니다.

## 1. 현재 상태 (2026-05-28)

| 항목 | 적용 여부 | 근거 |
|---|---|---|
| GPS 측위 | 미사용 | 펌웨어에 GPS 칩 미탑재 + 앱에 측위 로직 없음 |
| Wi-Fi 측위 | 미사용 | 펌웨어에 측위 로직 없음 — `iconia_*.cpp` 코드 검증 |
| BLE 비콘 측위 | 미사용 | 측위 비콘 미사용 |
| Android BLE 스캔 권한 | OS 정책상 위치 권한 사용 | 형식적 요건 — 실제 위치 미수집, [기기 연결 안내] 화면 명시 |
| IP 기반 위치 추정 | 미사용 | 운영 로그의 IP 는 보안·장애 대응 목적에만 활용 |

## 2. 본 정책 활성화 조건

다음 중 하나라도 도입 시 본 정책의 §3 이하 본문이 적용됩니다.

- 위치 기반 매장 안내 / 추천
- 디바이스 분실 추적 (인형의 위치 추적)
- 위치 기반 광고 / 마케팅
- 사용자 활동 영역 분석

## 3. (향후 적용 시) 수집 항목

| 항목 | 수집 방법 | 정밀도 |
|---|---|---|
| 위도·경도 | GPS 또는 Wi-Fi 측위 | 결정 후 갱신 |
| Wi-Fi SSID 목록 | (수집 시) Wi-Fi 측위 보조 | 결정 후 갱신 |
| 마지막 위치 시각 | 자동 | 분 단위 |

## 4. (향후 적용 시) 사용자 동의

- **동의 항목**: "위치정보 이용 동의" (선택)
- **거부 시 영향**: 위치 기반 기능 비활성. 기본 서비스는 정상 이용 가능.
- **철회 방법**: 앱 설정 > 동의 내역 관리 + OS 설정 > 위치 권한 회수

## 5. (향후 적용 시) 위치정보 사업자 신고 여부

「위치정보의 보호 및 이용 등에 관한 법률」 (위치정보법) 에 따라:

- **위치정보사업자 (제5조)**: 위치정보를 수집·저장·제공하는 사업 — 신고 대상
- **위치기반서비스사업자 (제9조)**: 위치정보를 이용하여 서비스 제공 — 신고 대상

본 서비스의 향후 위치 기능 도입 시 위 둘 중 하나에 해당하는지 검토 후 신고 절차를 진행합니다. 현재는 위치정보 미사용으로 비적용.
