# AI 이용 안내 / 면책 정책

## 1. AI 페르소나 사용 안내

ICONIA 서비스의 AI 페르소나는 다음 기술을 사용합니다.

- **Google Gemini API** — 응답 생성을 위한 대규모 언어 모델 (LLM)
- **자체 SOUL 엔진** — 페르소나 일관성, 5가지 욕구 → 8가지 신호 매핑, 4층 기억, 감정 온도 등 회사 자체 IP
- **RAG (Retrieval-Augmented Generation)** — 페르소나 별 기억·문서 기반 응답

## 2. 사용자에게 표시되는 안내 문구 (앱 내 [AI 이용 안내] 화면)

> ICONIA 의 AI 페르소나 응답은 인공지능이 생성한 것입니다.
>
> - 응답은 사실과 다르거나 부정확할 수 있으며, 의료·법률·금융 등 전문가 상담이 필요한 분야의 결정 근거로 사용하지 마세요.
> - 비밀번호·카드번호·주민등록번호 등 민감한 개인정보를 AI 와의 대화에 입력하지 마세요.
> - AI 페르소나 응답은 일관성을 위해 캐릭터화되어 있으며, 실제 인물이나 의료적·법률적 진단을 대체하지 않습니다.
> - AI 응답이 부적절하다고 판단되시면 [신고하기] 를 눌러 회사에 알려주세요.

## 3. 데이터 처리

| 데이터 | 처리 위치 | 보관 기간 | 마스킹 |
|---|---|---|---|
| 사용자 입력 메시지 | Google Gemini (US) + 회사 RDS (KR) | RDS 90일 / Gemini 정책에 따름 | PII redaction 후 저장·송신 |
| AI 응답 | 회사 RDS (KR) | 90일 | 그대로 보관 |
| 페르소나 컨텍스트 (기억) | AWS EFS / S3 (KR) | 사용자 탈퇴 시 폐기 | 사용자별 격리 |
| 운영 로그 | CloudWatch (KR) | 30일 | redact.js 마스킹 |

## 4. 사용자 동의 항목

회원 가입 시 다음 동의를 받습니다.

- **(필수) 제3자 제공 동의 (Google Gemini)** — AI 응답을 위해 사용자 메시지를 Google 에 전송
- **(필수) 국외이전 동의** — 미국 region 으로 데이터 전송 (Gemini API)
- **(선택) 분석·서비스 개선 동의** — 익명 통계로 활용

## 5. 학습 데이터 사용

**회사 정책**:
- 회사는 사용자 대화 내용을 자체 AI 학습에 사용하지 **않습니다**.
- 사용자 대화는 Gemini API 호출 시 Google 의 정책에 따라 처리됩니다.
- 회사는 PII redaction 을 적용한 후에만 데이터를 저장·송신합니다.

**향후 자체 학습 데이터 활용 시**:
- 별도 동의 항목 추가 (필수 또는 선택)
- 사용자가 동의 거부 시에도 기본 AI 기능은 정상 이용 가능
- 별도 [개인정보 처리방침] 및 [AI 이용 안내] 갱신 → 재동의

## 6. AI 안전성 조치

| 항목 | 조치 |
|---|---|
| Prompt injection 방어 | 입력 sanitization + canary token (`3. AI/src/rag/canaryToken.js`) |
| 시스템 프롬프트 누출 방지 | 응답 검증 + canary detection 자동 회수 |
| 부적절한 응답 차단 | Gemini safety filter + 자체 검증 |
| PII 마스킹 | 요청 전 마스킹 (`3. AI/src/utils/redact.js`) |
| 환각 (hallucination) 완화 | RAG 기반 사실 응답 우선, 모를 때 "확실하지 않다" 응답 |
| 페르소나 일관성 | SOUL 엔진 + 응답 후 일관성 검증 |
| 부정 사용 차단 | rate limit + 사용자 단위 anomaly detection |

## 7. AI 장애 시 대응

| 시나리오 | 사용자 측 동작 | 운영팀 동작 |
|---|---|---|
| Gemini API timeout | in-character fallback 응답 (canned message) | 키 풀 상태 점검 |
| Gemini API rate limit | 키 풀 round-robin + cooldown | 신규 키 발급 또는 Tier 승급 |
| Gemini API outage | 페르소나 일관성 유지 + "응답이 잠시 지연됩니다" 안내 | incident response 절차 |
| 부적절한 응답 감지 | 응답 차단 + 페르소나 적합 사과 메시지 | 응답 검토 + 페르소나 조정 |
| 키 풀 전체 cooldown | in-character fallback + "AI 페르소나가 잠시 휴식 중입니다" | 즉시 신규 키 추가 + Tier 승급 검토 |

운영팀 runbook: `docs/operations/ai-provider-incident-runbook.md`

## 8. 책임 제한

- AI 응답은 사용자의 의사결정을 대체하지 않습니다.
- 회사는 AI 응답으로 인한 직접/간접 손해에 대해 고의 또는 중과실이 없는 한 책임을 지지 않습니다.
- 사용자는 AI 응답을 비판적으로 검토하시고, 중요한 결정은 전문가 상담을 받으시기 바랍니다.

## 9. 신고 및 피드백

AI 응답이 부적절하거나 정책에 어긋난다고 판단되시면 [신고하기] 버튼으로 회사에 알려주세요. 회사는 신고 접수 후 영업일 2일 이내 검토하고 결과를 알려드립니다.

## 10. 사용자 권리

- AI 학습 데이터 사용 동의 철회 (현재 자체 학습 미사용)
- 제3자 제공 동의 철회 → AI 페르소나 응답 기능 비활성
- AI 대화 내역 열람 → 앱 설정 > AI 기록
- AI 대화 내역 삭제 → 앱 설정 > AI 기록 관리 > 전체 삭제 (90일 retention 보다 빠르게)

## 11. 정책 변경
