# AI 이용 안내

> 💡 **핵심 요약** — ICONIA AI 페르소나는 Google Gemini / Anthropic Claude / OpenAI GPT 등 다중 LLM 기반입니다. AI 응답은 부정확할 수 있으며 의료·법률·금융 판단의 근거로 사용할 수 없습니다. 민감한 개인정보는 입력하지 마세요.

## 1번 항목 — AI 페르소나 기술 구성

| 구성 요소 | 설명 |
|---|---|
| Google Gemini API | 응답 생성을 위한 대규모 언어 모델 (기본 LLM) |
| Anthropic Claude API | 응답 생성을 위한 대규모 언어 모델 (다중 LLM) |
| OpenAI GPT API | 응답 생성을 위한 대규모 언어 모델 (다중 LLM) |
| 자체 페르소나 엔진 | 페르소나 일관성, 욕구·신호 매핑, 기억 관리, 감정 표현 등 회사 자체 기술 |
| 검색 기반 응답 | 페르소나별 기억·문서를 검색해 응답에 활용 |

> ℹ️ 회사는 서비스 품질 및 가용성 확보를 위해 위 LLM 중 하나 이상을 상황에 따라 선택하여 사용합니다. 「AI 기본법」 §27에 따라 AI 응답임을 앱 내 UI에 명시합니다.

## 2번 항목 — 앱 내 AI 이용 안내 문구 (AI 기본법 §27·§29 준수)

> ⚠️ **AI 이용 전 반드시 확인하세요**
>
> ICONIA의 AI 페르소나 응답은 인공지능(Google Gemini / Anthropic Claude / OpenAI GPT)이 생성한 것입니다.
>
> 「인공지능 발전과 신뢰 기반 조성 등에 관한 법률」 제27조(AI 표시의무) 및 제29조(AI 생성물 표시)에 따라, AI 응답임을 채팅 화면에 표시합니다.
>
> - 응답은 사실과 다르거나 부정확할 수 있으며, **의료·법률·금융 등 전문가 상담이 필요한 분야의 결정 근거로 사용하지 마세요.**
> - 비밀번호·카드번호·주민등록번호 등 민감한 개인정보를 AI와의 대화에 입력하지 마세요.
> - AI 페르소나 응답은 캐릭터화되어 있으며, 실제 인물이나 의료적·법률적 진단을 대체하지 않습니다.
> - AI 응답이 부적절하다고 판단되시면 [신고하기]를 눌러 회사에 알려주세요.

## 5번 항목 — 학습 데이터 사용 정책

> ✅ 회사는 사용자 대화 내용을 자체 AI 학습에 사용하지 않습니다.

- 사용자 대화는 Gemini API 호출 시 Google 정책, Claude API 호출 시 Anthropic 정책, GPT API 호출 시 OpenAI 정책에 따라 처리됩니다.
- 회사는 개인정보 마스킹을 적용한 후에만 데이터를 저장·송신합니다.
- 자체 학습을 시작하게 될 경우 별도 동의 항목 추가 + 재동의 절차를 거칩니다.

## 6번 항목 — AI 안전성 조치

| 항목 | 조치 |
|---|---|
| 악의적 입력 방어 | 입력값 정제 + 모니터링 신호 적용 |
| 시스템 지시문 보호 | 응답 검증 + 누출 감지 시 자동 회수 |

| 부정 사용 차단 | 요청 한도 제한 + 이상 행동 감지 |

## 7번 항목 — AI 장애 시 대응

| 시나리오 | 사용자 측 표시 | 운영팀 조치 |
|---|---|---|
| Gemini / Claude / GPT API 응답 지연 | 대체 응답 표시 | 외부 API 키 상태 점검, 타 LLM 자동 전환 |
| API 요청 한도 초과 | 여러 LLM 자동 전환 + 쿨다운 | 신규 키 발급 또는 사용 등급 승급 |
| 특정 LLM API 장애 | 다른 LLM으로 자동 전환 | 회사 내부 장애 대응 절차 진행 |
| 모든 LLM API 장애 | "응답이 잠시 지연됩니다" 안내 | 긴급 복구 절차 진행 |
| 부적절한 응답 감지 | 응답 차단 + 사과 메시지 | 응답 검토 + 페르소나 조정 |
| 모든 API 키 쿨다운 | "AI 페르소나가 잠시 휴식 중입니다" 안내 | 신규 키 즉시 추가 |

## 8번 항목 — 책임 제한

- AI 응답은 사용자의 의사결정을 대체하지 않습니다.
- 회사는 AI 응답으로 인한 직접/간접 손해에 대해 고의 또는 중과실이 없는 한 책임을 지지 않습니다.
- 사용자는 AI 응답을 비판적으로 검토하시고, 중요한 결정은 전문가 상담을 받으시기 바랍니다.
