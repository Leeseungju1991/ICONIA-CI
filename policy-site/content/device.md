# 기기 연결·데이터 안내

> 💡 **핵심 요약** — ICONIA 인형은 블루투스 5.0 + Wi-Fi 2.4GHz IoT 기기입니다. 마이크·위치 기능은 없으며, 카메라는 손길 1회당 1장만 촬영합니다. Wi-Fi 비밀번호는 인형 안에만 저장되며 서버로 전송되지 않습니다.

## 1번 항목 — 기기 사양

| 항목 | 값 |
|---|---|
| 무선 통신 | 블루투스 5.0 + Wi-Fi 2.4GHz |
| 카메라 | 손길(터치) 시 1장 캡처 — 임시 메모리 처리 후 즉시 폐기 |
| 마이크 | **미탑재** |
| 위치 기능 | **미탑재** |
| 인증 | KC 적합성평가 (양산 출하 전 진행) |

## 2번 항목 — 기기 연결 절차

> ℹ️ ICONIA 앱과 인형을 처음 연결할 때 다음 3단계를 거칩니다.

**1단계 — 블루투스 권한 동의**
앱이 인형을 블루투스로 검색·연결하기 위해 필요합니다.

**2단계 — 위치 권한 (Android 전용)**
Android OS의 블루투스 스캔은 운영체제 정책상 위치 권한을 함께 요구합니다. **인형의 위치나 사용자의 위치를 수집하지 않습니다.** 운영체제 정책상의 형식적 요건입니다.

**3단계 — Wi-Fi 비밀번호 입력**
인형이 가정 Wi-Fi에 연결하기 위해 Wi-Fi 이름과 비밀번호를 블루투스를 통해 인형에 직접 전송합니다.

### Wi-Fi 비밀번호 저장 정책

| 저장 위치 | 방식 |
|---|---|
| 앱 (사용자 휴대폰) | 기기 보안 저장소에 암호화 저장 |
| 인형 | 인형 내부 저장소에 암호화 저장 |
| 회사 서버 | **전송·저장 안 함** |

## 3번 항목 — 기기에서 수집·송신하는 데이터

| # | 데이터 | 기기 측 보관 | 서버 전송 |
|---|---|---|---|
| 1 | 카메라 사진 (손길 시 1장) | 임시 메모리만 — 업로드 직후 즉시 폐기 | 암호화 통신, 회사 클라우드(대한민국 리전) |
| 2 | 터치 이벤트 (좌/우/타이밍) | 중복 방지 키만 보관 | 이벤트 메타와 함께 송신 |
| 3 | 배터리 잔량·상태 (6단계) | 마지막 값만 보관 | 이벤트 메타와 함께 송신 |
| 4 | Wi-Fi 이름/비밀번호 | 암호화 저장 | **송신 없음** |
| 5 | 기기 식별자 | 안전 영역에서 일회성 식별값으로 변환 | 페어링·이벤트 시 |
| 6 | 부팅 진단 정보 | 임시 보관 | 부팅 직후 |
| 7 | OTA 업데이트 진행 정보 | 내부 큐 | 업데이트 진행 중 |
| 8 | 에러 보고 | 임시 카운터 | 일정 횟수 제한하여 송신 |
| 9 | 소프트웨어 버전·하드웨어 개정판·모델명 | 기기 정보 | 매 요청 헤더 |

## 4번 항목 — 수집하지 않는 데이터

> ✅ 다음 항목은 하드웨어 설계 차원에서 수집·처리하지 않습니다.

| 항목 | 이유 |
|---|---|
| 위치 정보 (GPS/Wi-Fi 측위) | 위치 측위 기능 미탑재 |
| 마이크/음성 | 마이크 미탑재 |
| 연속 영상/항시 녹화 | 저전력 정책 — 손길 시 1장만 |
| 얼굴 인식·생체 정보 | 인식·처리하지 않음 |
| 결제·금융 정보 | 기기가 처리하지 않는 범위 |
| Wi-Fi 비밀번호 서버 전송 | 기기 로컬에만 저장 |

## 4-1번 항목 — 카메라 촬영 LED 및 영상정보 운영기준

「개인정보 보호법」 및 관련 규정에 따라, 인형 카메라 촬영 시 사용자가 인지할 수 있도록 다음과 같이 운영합니다.

| 항목 | 내용 |
|---|---|
| 촬영 LED | 카메라 셔터 작동 시 인형 LED가 짧게 점등(흰색/청색)하여 촬영 사실을 표시 |
| 촬영 방식 | 손길(터치) 1회당 정지 이미지 1장만 — 항시 녹화·연속 영상·원격 임의 촬영 없음 |
| 인형 내 보관 | 없음 — 암호화 업로드 직후 즉시 폐기 |
| 서버 보관기간 | 회사 클라우드(대한민국 서울 리전) 보관 후 「개인정보 처리방침」 3번 항목에 따라 파기 |
| 파기 방법 | 보관기간 만료 또는 사용자 삭제 요청 시 복구 불가능한 방식으로 즉시 파기 |
| 항시 녹화 여부 | 없음 — 하드웨어·소프트웨어 차원에서 항시 녹화 불가 |
| 비활성화 | 앱 설정 > 카메라 기능 OFF (비활성화 후 채팅 기능은 정상 사용 가능) |

## 5번 항목 — 안전 사용 안내

> ⚠️ 다음 사항을 반드시 지켜주세요.

- **분해·개조 금지** — 보증 및 KC 안전 인증이 무효화됩니다.
- **과열·연기·이상 소음 발생 시** 즉시 전원을 차단하고 고객지원(web@soomkorea.com)에 문의하세요.
- **어린이 손에 닿지 않도록** 보관하세요 (성인 대상 제품입니다).
- **야외·자동차 내부·고온 환경(60°C 이상)** 사용 금지 — 가정 내 사용 전용 설계.
- **배터리 충전** 시 정품 충전기·케이블만 사용하세요.

## 6번 항목 — 소프트웨어 자동 업데이트

- 회사는 보안·기능 개선을 위해 기기 소프트웨어를 자동으로 업데이트합니다.
- 사용자가 거부할 경우 일부 기능 사용이 제한될 수 있습니다.
- 업데이트 실패 시 자동으로 이전 안정 버전으로 복귀하며, 사용자 데이터는 영향받지 않습니다.

## 7번 항목 — 기기 폐기·소유권 이전 절차

이용자가 기기를 폐기하거나 다른 사용자에게 양도할 경우, 다음 절차로 사용자 데이터가 안전하게 삭제됩니다.

1. Wi-Fi 이름과 비밀번호 삭제
2. 블루투스 페어링 정보 삭제
3. 기기 식별값 재생성
4. 진단 정보 삭제
5. 세션 정보 초기화
6. 고객지원 채널에 삭제 완료 보고
7. 새 사용자가 페어링할 수 있도록 대기 상태로 전환

기기의 공장 출하 시 부여된 안전 식별자는 기기 정체성 보존을 위해 삭제 대상이 아닙니다.
