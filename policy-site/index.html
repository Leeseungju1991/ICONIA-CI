다운로드 · 설치

처음 한 번만, 5단계로 충분해요

사이트에서 바로 받는 방식이라 안내 절차가 살짝 다릅니다. 천천히 따라와 보세요.

인형 개봉 → 앱 설치 → 블루투스 → Wi-Fi → 앱 사용

① 앱 받기

② 설치하기

  1. 1
    APK 파일 받기 (Android)

    위 [Android] 버튼을 누르면 자동으로 다운로드 폴더에 저장됩니다.

    파일명: iconia-latest.apk (약 40 MB)

  2. 2
    "출처를 알 수 없는 앱" 한 번만 허용

    Android: 설정 → 보안 → "출처를 알 수 없는 앱 설치 허용" 또는 다운로드 직후 알림에서 "허용" 탭

    iOS: 설정 → 일반 → VPN 및 기기 관리 → "(주)숨코리아" 신뢰

  3. 3
    APK 파일 탭 → 설치

    다운로드 폴더에서 iconia-latest.apk 를 누르면 설치 화면이 뜹니다.

    "설치" 버튼만 누르면 자동으로 진행돼요.

  4. 4
    회원가입 → 인형 개봉

    앱을 열고 이메일로 회원가입 (1분 소요)

    인형 박스에서 인형을 꺼내고 전원 버튼을 길게 눌러 켜기

  5. 5
    블루투스 페어링 → Wi-Fi 연결 → 시작

    앱이 자동으로 주변 인형을 찾아 페어링 요청해요. "허용" 탭

    Wi-Fi 비밀번호를 한 번 입력하면 인형이 자동으로 연결돼요. 끝! 인형을 살짝 만져보세요.
